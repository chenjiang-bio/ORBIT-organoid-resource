# Literature Agent

Multi-agent literature extraction system for ORBIT.

## Scope

This module will host the agent workflow that extracts and standardizes organoid culture-protocol and related information from scientific publications into structured records for ORBIT.

## Status

Scaffold only. Implementation will be contributed by collaborators.

## Dependencies

Runtime tool access for this agent is provided by the MCP services in [`../literature-mcp/`](../literature-mcp/). Configure and start those services before running the agent when both modules are available.
